# yt6801 Linux 驱动 — LoongArch (loong64)

<p align="center"><a href="README.md">English</a> | <a href="README-zh.md">中文</a></p>

[裕太微电子 (Motorcomm)](https://www.motor-comm.com) **yt6801** 以太网卡驱动的预编译内核模块（`yt6801.ko`），适用于 *
*LoongArch (loong64)** 架构 **Debian 14 (deb14)** 系统。

## 概述

- **厂商：** 裕太微电子 (Motorcomm / 裕太微电子)
- **目标平台：** LoongArch 64 位 (loong64)，Debian 14
